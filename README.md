# React redux shopping demo app

- list products 
- add product to cart
- manager cart (price , items)

## Minimum requirements

- [Node 6.9.1][nodejs]


## Running the Application

- Running Express Application

```shell
$ npm run dev
```

- Running frontend application with webpack

```shell
$ npm run webpack
```
